이미지 기반 OCR과 AI 설명 생성을 결합한 Windows 유틸리티 프로그램

프로젝트 유형
Windows 유틸리티
개발 언어
Python
GUI 프레임워크
PySide6
핵심 기술
OCR + AI

기획

이미지 속 텍스트를 추출하는 과정은 보통 별도의 OCR 프로그램 설치, 경로 설정, 언어 데이터 구성 등 여러 준비 단계를 요구한다. 특히 일반 사용자는 명령어 기반 도구를 직접 다루기 어렵고, 단순히 화면 속 텍스트를 빠르게 확인하고 싶을 때도 복잡한 설정 과정이 필요하다.

이러한 불편을 줄이기 위해 본 프로젝트에서는 이미지 선택만으로 OCR을 수행하고, 추출된 텍스트에 대해 AI 기반 요약과 설명까지 제공하는 GUI 프로그램을 제작하였다. 목표는 사용자가 복잡한 환경 설정 없이도 즉시 사용할 수 있는 직관적인 Windows 유틸리티를 만드는 것이다.

구성물

주요 기능

  • 이미지 파일 불러오기
  • OCR 실행 및 텍스트 추출
  • 추출 텍스트 표시
  • AI 기반 요약 생성
  • AI 기반 설명 생성
  • 결과 저장 기능

프로그램 구성 요소

  • PySide6 기반 GUI 화면
  • Tesseract OCR 엔진
  • Pillow 기반 이미지 처리
  • AI API 연동 모듈
  • Windows 배포를 위한 EXE 패키징 구조
Python PySide6 Pillow pytesseract Requests PyInstaller

실현 방법

프로그램은 Python으로 구현하였으며, GUI는 PySide6를 사용하여 구성하였다. 사용자가 이미지를 불러오면 Pillow를 통해 이미지를 처리하고, pytesseract를 이용하여 내부적으로 연동된 Tesseract OCR 엔진으로 텍스트를 추출한다.

OCR 결과는 프로그램 내부 텍스트 영역에 출력되며, 이후 AI API를 호출하여 추출 텍스트의 요약과 설명을 생성하도록 구현하였다. 이를 통해 단순 텍스트 인식에서 끝나지 않고, 사용자가 내용을 더 쉽게 이해할 수 있도록 보조하는 기능까지 제공한다.

또한 배포 편의성을 위해 Tesseract 실행 파일과 언어 데이터를 프로젝트 내부의 third_party 폴더에 포함시키는 구조를 적용하였고, PyInstaller를 사용하여 Python이 설치되지 않은 환경에서도 실행 가능한 Windows용 프로그램으로 패키징하였다.

동작 흐름
이미지 불러오기 → 이미지 전처리 → OCR 실행 → 텍스트 출력 → AI 요약 생성 → AI 설명 생성 → 결과 저장

결과

사용자는 GUI에서 이미지를 선택한 뒤 OCR 실행 버튼을 눌러 텍스트를 추출할 수 있으며, 동일한 화면에서 AI 설명 생성 기능을 통해 추출 텍스트의 요약과 해설을 확인할 수 있다. 이를 통해 기존의 명령어 기반 OCR 환경보다 훨씬 직관적인 사용자 경험을 제공하도록 구현하였다.

프로그램은 GUI 실행, 이미지 로드, OCR 처리, 결과 표시까지의 기본 기능을 수행하도록 구성되었으며, Windows 환경에서 배포 가능한 형태로 정리하였다.

특장점

  • OCR 엔진을 포함한 구조로 설계하여 별도 설치 없이 실행 가능한 방향으로 구현하였다.
  • GUI 기반 인터페이스를 제공하여 비전문가도 쉽게 사용할 수 있도록 하였다.
  • 이미지 내 텍스트 추출뿐 아니라 AI 기반 설명 기능을 결합하여 활용도를 높였다.
  • Windows 환경에서 실행 가능한 유틸리티 형태로 배포할 수 있도록 구성하였다.

한계점

  • OCR 정확도는 입력 이미지의 해상도, 대비, 글자 크기에 따라 달라질 수 있다.
  • 복잡한 표 구조나 수식, 특수한 배치의 문서는 인식 정확도가 낮을 수 있다.
  • AI 설명 기능은 외부 API 사용 환경에 따라 동작 여부가 달라질 수 있다.
  • 현재 버전은 기능 구현에 초점을 맞춘 단계로, 예외 처리와 후처리 기능은 추가 개선이 필요하다.
